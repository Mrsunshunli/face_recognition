# face_recognition
-------------------------------------------------------
一：项目介绍
这个项目是人脸识别与检测，可以通过图片来检测，可以通过摄像头来实时检测。这个代码的缺陷还是很大的，首先你用摄像头进行截取照片，
照片的相识度很大，所以需要数据争强，还有受到光照等因素的影响也很大。大家如果有什么更好的办法，可以指出，非常感谢。
--------------------------------------------------------------------------------------
